# Lucifer Fashion and Trends (LFT) Online Shopping App

Welcome to the Lucifer Fashion and Trends (LFT) Online Shopping App repository! This repository houses the codebase for our React Native mobile application, powered by a Node.js backend and MongoDB database. LFT offers users a seamless shopping experience for the latest fashion trends and styles.

## Features

- Browse and search for a wide range of fashion products.
- View detailed product information, including images, descriptions, and pricing.
- Add products to the cart and manage cart items.
- Secure user authentication and registration.
- Checkout functionality with support for payment processing.
- Order tracking to monitor the status of purchases.
- User profile management.

## Technologies Used

- **Frontend**: React Native
- **Backend**: Node.js
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **Payment Integration**: Razorpay Payment Gateway

## Getting Started

To get a local copy of the project up and running, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Akki4460/Ecommerce.git

2. **Install Dependencies**:
   ```bash
   cd LFT
   npm install

3. **Configure environment variables**:
  - Create a .env file based on the provided .env.example template.
  - Fill in the necessary environment variables such as MongoDB connection URI, JWT secret, etc.

3. **Run the backend server**:
   ```bash
   npm start

3. **Run the react-native app**:
   ```bash
   # Ensure you have set up your development environment for React Native
   npx react-native run-android
   # or
   npx react-native run-ios
